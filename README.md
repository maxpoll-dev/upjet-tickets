# ЗАПУСК

### Команды bash

- Скопировать конфиг - ``cp .env.example .env``
- Поднять докер - ``make dev``
- Запустить миграции -  ``make migrate``
- Запустить сидэр -  ``make seed``
- Открыть http://localhost:3000/
